# 自定义样式

框架在CSS层面，使用了CSS变量对一些关键部分进行可编程配置，这样方便自定义主题，你可以直接在浏览器查看然后覆盖对应的变量值，也可以参考w.css文件。以下是w.css文件中用到的CSS变量：

```css
:root {
  --w-alert-color: #4a4a4a;
  --w-alert-bg-color: #f6f7f9;
  --w-alert-color: #4a4a4a;
  --w-alert-close-bg-color: #f5f6f7;
  
  --w-tabbar-color: #4a4a4f;
  --w-tabbar-bg-color: #fdfdfd;

  --w-page-bg-color: #fcfcff;

  --w-slider-bg-color: #fcfcff;
  --w-slider-close-bg-color: rgba(21,22,25,0.32);
  --w-common-shadow: 0px 0.1rem 1.2rem 0.1rem rgba(15, 15, 15, 0.1),inset 0px -0.1rem 0px 0px rgba(15,15,15,0.1);

  --w-notify-color: #f1f2f3;
  --w-notify-bg-color: rgba(62, 64, 68, 0.85);

  --w-notify-light-color: #4a4a4a;
  --w-notify-light-bg-color: rgba(240, 240, 240, 0.95);
  
  --w-prompt-color: #4a4a4a;
  --w-prompt-bg-color: #fafafd;

  --w-prompt-glass-bg-color: rgba(42,43,45, 0.033);

  --w-cover-bg-color: rgba(22,23,25, 0.15);
}
```
